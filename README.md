## 💡 2. Первичная Идея, Цель и Задачи
### Проблема

Владельцы пекарен и кофеен считают себестоимость в Excel. При изменении цен на продукты они забывают пересчитать рецепты, теряют маржу и работают в убыток.
### Решение

Веб-сервис, который автоматически пересчитывает себестоимость блюд при изменении цен на ингредиенты и предупреждает о падении маржи.
### Глобальная Цель

Стать инструментом №1 для учета себестоимости.
### Задачи на MVP (3 месяца)

1. Реализовать учет ингредиентов и истории цен
2. Сделать конструктор техкарт (рецептов)
3. Автоматический пересчет себестоимости в фоне
4. Личный кабинет с графиками маржинальности

| **Сильные стороны**                         | **Слабые стороны**                 |

## 🥊 5. Анализ Конкурентов

### Типология конкурентов

1. **Прямые:** Специализированные сервисы калькуляции (ChefCalc, Калькулятор-техкарт)
2. **Косвенные:** Товароучетные системы (МойСклад, 1C: Общепит)
3. **Основные (Статус-кво):** Excel / Google Таблицы / Бумажные журналы
4. **Отвалившиеся:** Сложные ERP для заводов, которые пытались зайти в малый бизнес

| Конкурент              | +                          | -                                                      | Почему плохо                             |
| ---------------------- | -------------------------- | ------------------------------------------------------ | ---------------------------------------- |
| Таблицы(Excel, google) | Бесплатно, гибкие          | Ручная работа, нет алертов                             | Высокий риск ошибки                      |
| 1С:ОБЩЕПИТ             | Полный учет, склад, касса  | Дорого, сложно, нужен специалист, громоздкий интерфейс | Избыточен для маленькой пекарни          |
| ChefCalc               | База рецептов, расчет КБЖУ | Платный, нет уведомлений об изменении                  | Фокус на нутрициологии, а не на финансах |
| МойСклад               | Удобный интерфейс, облако  | Дорого для старта                                      | Это склад, а не калькулятор рецептов     |

## 🗡️ 7. Новизна

### Почему мы?
1. **Проактивность**  
	Мы не просто считаем, мы **предупреждаем**. (Excel молчит, когда вы теряете деньги, мы нет).
2. **Простота**  
	Вход за 5 минут. Не нужно быть бухгалтером, чтобы понять интерфейс.
3. **Фокус на марже**  
	Мы показываем не просто цену, а **точку безубыточности** с учетом постоянных расходов (аренда, свет), что редкость в простых калькуляторах.
4. **История цен**  
	Единственный сервис, который хранит историю изменения цен и позволяет пересчитать себестоимость на любую дату.
